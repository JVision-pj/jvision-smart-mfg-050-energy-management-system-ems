# 能源管理管理艙

監控與管理工廠用電、用水、燃氣等能源消耗，優化能源使用效率之系統。

- 原始系統：能源管理系統（Energy Management System, EMS）
- 產業分類：智慧製造｜製造管理
- 使用部門：廠務、設備工程、永續發展部門、財務
- 工作流程：能耗數據採集 → 分項統計分析 → 異常耗能告警 → 節能對策執行 → 效益驗證追蹤
- AI 應用：AI預測能耗需求並優化排程避峰、智慧偵測異常耗能設備
- 主要 KPI：單位產品能耗、節能達成率、尖峰負載降低率
- 代表產品：Schneider Electric EcoStruxure、Siemens EnergyIP、Honeywell Energy Manager
